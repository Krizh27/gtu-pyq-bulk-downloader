# Directory Structure Explanation

This document explains what every folder in the project does. 

## 1. `/docs`
Contains all the documentation for the project, including the architecture, lifecycle flows, and historical migration reports. 

## 2. `/frontend`
This folder contains the entire React Single-Page Application. It runs on port `5173`.
- `package.json`: Lists the frontend dependencies (like React, Tailwind, Vite).
- `vite.config.js`: Configuration for the Vite bundler.

### `/frontend/src`
Contains all the custom source code for the browser application.
- **`/api`**: Contains the networking code. 
  - `custom-fetch.js`: A helper to standardize `fetch` requests.
  - `generated/api.js`: The React Query hooks and Zod schemas used to easily communicate with the backend.
- **`/components`**: Reusable pieces of UI.
  - `/ui`: Contains simple, unstyled components (like `Button.jsx`, `Input.jsx`) that have been customized with Tailwind CSS. These were initially generated by `shadcn/ui`.
- **`/hooks`**: Custom React hooks (like `useToast` for showing popups).
- **`/lib`**: Helper libraries and utility functions (like Tailwind class mergers).
- **`/pages`**: The main screens of the application (`home.jsx`, `download.jsx`, `about.jsx`).

## 3. `/backend`
This folder contains the Node.js / Express server. It runs on port `8080`.
- `package.json`: Lists the backend dependencies (like Express, Pino, Zod).
- `openapi.yaml`: A specification file defining exactly what the API endpoints look like and what data they expect.

### `/backend/src`
Contains all the logic for the API server.
- **`/api`**: Contains the Zod schemas used to validate incoming JSON requests against the `openapi.yaml` rules.
- **`/lib`**: General utilities (like the `logger.js` configuration).
- **`/routes`**: Contains the actual API endpoints that respond to frontend requests.
  - `health.js`: A simple endpoint returning `{ status: "ok" }` to verify the server is alive.
  - `pyq/index.js`: The core business logic endpoint that accepts subject codes and generates candidate GTU PDF URLs.
